# Security Policy

## Reporting

Do not report security issues through public GitHub issues.

Use GitHub Security Advisories for this repository when available:

- `https://github.com/FengYunCalm/opencode-peer-session-relay/security/advisories/new`

If that route is unavailable, contact the repository owner through a private channel listed on GitHub.

## What to report

Please report:

- room, thread, message, or workflow isolation failures
- auth or delivery issues that expose data across sessions unexpectedly
- committed secrets, credentials, or unsafe local defaults
- replay or persistence bugs with security impact

## Supported versions

Only the latest code on the default branch is considered supported.
